# Hardware Design

This folder contains the native source files for the Automatic Smart Light project, designed using **EasyEDA Pro**.

## 📁 Files
* **pcb1.epcb**: The primary project source file for EasyEDA Pro. This contains the full schematic, PCB layout, and design settings.

## 🛠 Usage
* **Editing**: To view or modify the design, open the `pcb1.epcb` file directly within the EasyEDA Pro environment.
* **Compatibility**: If you do not have EasyEDA Pro installed, please refer to the files in the `/Documentation` folder (PDFs/PNGs) for a visual representation of the design.

## ⚙️ Design Specifications
* **Tools**: EasyEDA Pro
* **Workflow**: Schematic capture followed by custom PCB routing[cite: 1]
