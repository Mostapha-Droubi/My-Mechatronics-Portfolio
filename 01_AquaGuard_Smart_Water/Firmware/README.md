# Firmware: AquaGuard

This folder contains the source code for the AquaGuard system.

## Requirements
* [Arduino IDE](https://www.arduino.cc/en/software)
* Libraries:
    * [Blynk Library](https://github.com/blynkkk/blynk-library)
    * [ESP8266 Board Manager](https://github.com/esp8266/Arduino)

## Configuration
Before uploading, ensure you update the `config.h` file with your specific:
- Wi-Fi SSID and Password.
- Blynk Auth Token.
- Calibration offsets for pH and Turbidity sensors.
